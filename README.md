🎯 Sobre o Projeto

O Sistema de Gestão FastAPI é uma API REST moderna e robusta desenvolvida para gerenciar clientes e seus aparelhos eletrônicos. O projeto foi construído com foco em performance, escalabilidade e facilidade de manutenção, seguindo os padrões e melhores práticas da comunidade Python.
✨ Principais Características

    🚀 Performance: Baseado em FastAPI, uma das frameworks mais rápidas disponíveis

    📊 Documentação Automática: Swagger UI e ReDoc gerados automaticamente

    🔒 Validação Robusta: Schemas Pydantic para validação de dados

    🏗️ Arquitetura Limpa: Separação clara de responsabilidades

    🔗 Relacionamentos: Sistema completo de foreign keys e relacionamentos

    💾 Persistência: PostgreSQL com SQLAlchemy ORM

    ⚡ Assíncrono: Suporte nativo a operações assíncronas

🎁 Funcionalidades
👥 Gestão de Clientes

    ✅ Cadastro de novos clientes

    ✅ Listagem de todos os clientes

    ✅ Busca por cliente específico

    ✅ Atualização de dados do cliente

    ✅ Exclusão de clientes

    ✅ Cadastro de aparelhos por cliente

    ✅ Listagem de todos os aparelhos

    ✅ Busca por aparelho específico

    ✅ Atualização de dados do aparelho

    ✅ Exclusão de aparelhos

    ✅ Validação de número de série único
